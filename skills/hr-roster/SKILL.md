# /hr-roster

로스터 **현황표**만 보여준다. 판정도 권고도 방향도 없다 — 사실만. (판정·개편 권고가 필요하면 `/hr-review`.)

## 절차

1. 이 프로젝트의 worklog를 읽는다:
   `~/.claude/plugins/data/hr-plugin/<cwd를 슬래시→대시로 바꾼 이름>/worklog.jsonl`
   - 없거나 비면 → "받아 적힌 교대가 없다"고 알리고 명단만 보여준다.

2. `agents/*.md`의 정의된 역(직무)을 읽는다.

3. 표로 낸다 — **역 | 교대 수 | 마지막 교대(ts) | 최근 task 예시**.
   - **명단 ∖ worklog에 나온 역 = 안 불린 역**은 "교대 0"으로 따로 묶어 표시한다.
   - 교대 수 많은 순으로 정렬.

4. 끝. **판정·권고하지 않는다** — 사실만. (해석이 필요하면 `/hr-review`로.)
