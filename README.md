# Agent HR

멀티 에이전트 조직의 로스터를 적정 규모로 검토하는 인사(HR) 플러그인. 유휴 역 정리, 과부하 역 분산, 채용을 권고한다. 권고만 하고 실행은 사람이 승인한다.

상태: 초기(요구사항/스켈레톤 단계). HR 판단 로직은 아직 채우는 중이다.

## 설치

```
/plugin marketplace add rara-band/hr-plugin
/plugin install hr-plugin@rara-band
```

## 쓰는 법

`/hr-review` 로 조직 로스터를 검토한다. (스켈레톤 단계라 동작 본문은 아직 비어 있다.)
